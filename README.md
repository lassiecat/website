## Development

### Local Development

```bash
# Clone the repository
git clone https://github.com/lassiecat/website.git
cd website

# Open in browser
open index.html
```

### Deployment

This site is deployed automatically via GitHub Pages.

1. Push changes to main branch
2. GitHub Actions deploys to github.io
3. Site live at: https://lassiecat.github.io

## Technologies

- Pure HTML/CSS (no frameworks)
- GitHub Pages hosting (free)
- Custom domain ready
